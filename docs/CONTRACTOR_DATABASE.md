# Contractor Database Guide

Manage your contractor leads database.

## Overview
The Contractors page provides a full CRM for managing contractor leads collected via scraping.

## Views
- **Table View** — Compact, sortable list of all contractors
- **Cards View** — Visual card layout with key details

## Status Pipeline
- **New** — Freshly scraped, not yet contacted
- **Contacted** — Initial outreach made
- **Qualified** — Meets criteria, worth pursuing
- **Converted** — Became a customer
- **Lost** — No longer in pipeline

## Filtering
Use the search bar and status dropdown to filter contractors.
Reset filters with the refresh button.

## Scoring
Each contractor has a score (0-100) based on:
- Data completeness
- Business size indicators
- Category match
- Geographic relevance

## Export
Export to CSV, Excel, or PDF via the Export button.
